## Estimation and system identification

**Before you start:** Firstly, you need to watch the **first five videos** from [this playlist](https://www.youtube.com/playlist?list=PLXBJk7WTnAgWNib_2rO6EKZ0DiTRfbtSJ) on YouTube. You also need to have completed this [W3C Python tutorial](https://www.w3schools.com/python/) (excluding classes and file handling), including [this numpy tutorial](https://www.w3schools.com/python/numpy/default.asp) and [this matplotlib tutorial](https://www.w3schools.com/python/matplotlib_intro.asp).

### Basic setup

For detailed guidelines on how to set up VS Code to run these Python notebooks, read [these instructions](./docs/getting-started.md).

In you have already installed VS Code (or some other IDE), Python, the VS Code Python extension, and venv, then:
- Create a virtual environment (recommended: Python 3.13) for this project
- Install all packages with 
```bash
pip install -r requirements.txt
```
